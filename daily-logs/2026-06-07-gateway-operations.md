### 3. กระบวนการขาเข้า (Inbound) — 5 ประเภท

| ประเภท | หน้าที่ |
|--------|---------|
| Oversize | พัสดุที่เข้าเครื่องอัตโนมัติไม่ได้ (ผลรวม 3 ด้าน > 100 ซม. / หนัก > 50 กก.) *(แก้ไข 12 มิ.ย.: วัดผลรวม 3 ด้าน ไม่ใช่ด้านเดียว)* |
| โหลด | รับพัสดุขึ้นสายพาน |
| ท้ายราง | กระจายพัสดุ Oversize/VIP/เสียหาย ไปยังแผนกต่างๆ |
| ปัดงาน | คัดแยกพัสดุตาม Line (5 เส้น เส้นละ 7 คน) |
| DWS | ชั่ง-วัด-สแกนขนาดอัตโนมัติ |

**ขั้นตอนหลักขาเข้า:**
1. ตรวจสอบใบมอบหมายงาน (ชื่อคนขับ / ทะเบียน / สายรัด 3 เส้น)
2. ลงคิว คีย์ข้อมูล ปล่อยรถเข้าช่องโหลด
3. โหลดพัสดุผ่าน DWS

**DWS — ข้อควรระวัง:**
- วางพัสดุบนสายพานห่าง ≥ 10 ซม.
- ห้าม Label คว่ำหน้าลง (ระบบหยุดทันที)
- พัสดุ 2 Label → ส่ง Packing ทันที
- VIP brands ห้ามขึ้นเครื่อง: Oppo, Vivo, เดอะมอลล์, AIS, Watson, Permelo, DIY, Realme

### 4. Auto Packing / บ้านซอง

แบ่งเป็น **4 วง**:

| วง | จำนวนเครื่อง | Drop (ช่อง) | รถไฟ (ขบวน) |
|----|-------------|-------------|-------------|
| A | 24 | 424 | 758 (2 ชั้น) |
| B | 24 | 424 | 758 (2 ชั้น) |
| C | 24 | 428 | 780 (2 ชั้น) |
| D | 36 (ใหญ่สุด) | 384 | 792 (2 ชั้น) |
| Flip Tray | อยู่บริเวณวง D | 152 | 354 |

**Flip Tray** — สำหรับพัสดุทรงกลม ทรงกระบอก กลิ้งได้ หรือขึ้นเครื่องปกติไม่ได้

**เงื่อนไขพัสดุเข้าบ้านซอง:**
- น้ำหนัก ≤ 3 กก.
- ด้านยาวสุด ≤ 30 ซม. (ทั่วไป) หรือ ≤ 70 ซม. (ยาวบาง)
- ผลรวม 3 ด้าน ≤ 90 ซม.

**ขั้นตอนบ้านซอง:** เทกระสอบ → Auto Packing → รัดกระสอบ

**ประเภทกระสอบที่รับได้:**
- A1 / A0 → ปลายทาง กทม. + ปริมณฑล (สาขา)
- AAA → ปลายทาง ตจว. (GW)
- D1 Label → คัดแยกบ้านย่อย

**กระสอบ 4 ประเภท:** ซิปแดง / เขียว / น้ำเงิน / แดง CV
