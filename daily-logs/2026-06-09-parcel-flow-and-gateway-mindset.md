## What I Learned

### 1. สถานะสำคัญที่ต้องสังเกต (3 สถานะหลัก)

| สถานะ | ความหมาย |
|-------|---------|
| พัสดุมาถึง | พัสดุ scan ถึงจุดนั้นแล้ว |
| พัสดุอยู่ที่ | พัสดุยังค้างอยู่ที่จุดนั้น ยังไม่ถูกส่งต่อ |
| พัสดุส่งออก | พัสดุออกจากจุดนั้นไปแล้ว |

---

### 2. ขั้นตอนพัสดุทั้งหมดแบบละเอียด

```
รับพัสดุ
→ สแกนส่งพัสดุ
→ สแกนเปิดตู้
→ พัสดุถึง Gateway
→ สแกนบรรจุกระสอบ
→ สแกนส่งพัสดุขึ้นรถ
→ สแกนเปิดตู้
→ สแกนส่ง
→ สแกนลงรถ
→ สแกนนำจ่าย
```

---

### 3. Key Takeaway — Mindset การมอง Gateway

เวลาลงไปดูการทำงาน Gateway ให้มองให้ครบ 4 มุม:

1. **ทำอะไร** — หน้าที่และกระบวนการจริงๆ คืออะไร
2. **มีปัญหาอะไร** — ปัญหาที่เกิดขึ้นจริงในงาน
3. **ต้องการอะไร** — ความต้องการของคนทำงาน
4. **แนวทางแก้ไขยังไง** — ข้อเสนอแนะเพื่อปรับปรุง
